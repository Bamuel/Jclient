Java CLIENT SERVER
========
`by Samuel Levin`

Pokemon Journey is a simple JSWING application where there are 2 buttons

![MAIN APPLICATION](https://i.imgur.com/sArpRNF.png)



Features
--------

- It has a JBUTTON
- It has a JPANNEL
- It has a CLIENT SERVER
